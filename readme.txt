Tiny plugin that prints the Google Tag Manager scripts in the head and body of a site.

== Description ==

*Lean & Mean Google Tag Manager* is an extremely small and fast plugin that integrates Google Tag Manager into your website, and does nothing more. No unnecessary bloat or unwanted features.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress.
1. Go to 'Settings->General' and add your Google Tag Manager Container ID to the relevant field.
1. Click the 'Save Changes' button.
